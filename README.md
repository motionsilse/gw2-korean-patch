# 길드워2 한글 패치

본 한글 패치는 비공식 사용자 제작 패치입니다.


사용자 DB 수집과 번역 적용은 켜고 사용하시는걸 추천드립니다.

길드워2의 입문자 : 전체 체크 모두 번역 추천 or 아이템 이름만 체크해제


길드워2 기존 플레이어거나 업적 및 아이템등 위키 사용을 위한 플레이어 : 전체 체크하고 고유명사만 체크해제

그밖에 사용자입맛에 맞게 설정해서 사용가능.

길드워2는 14년이 된 게임으로 방대한 DB를 자랑합니다. 최대한 많은 DB를 번역했으나, 번역되지 않은 문구가 있을수 있습니다.
그래서 사용자 번역 이라는 기능을 활성화하면, 미번역된 문구를 바로 실시간으로 번역할수 있습니다.

사용자 DB 수집이란? 게임중에 번역되지 않았던 영어 원문을 그 즉시 수집하여 백그라운드로 번역합니다. (약1초 소요)
즉 사용자 번역을 켜 놓게 되면, 방금 대화했던 NPC와의 영어 지문, 클릭했던 오브젝트 영어 설명등을 다시 클릭하면
실시간으로 한글로 번역되어 있는걸 보실 수 있습니다.

단, 신규 문자열 자동 수집을 켜놓게 되면 맵이동 로딩이 느려질 수 있습니다.


문제가 발생하면 패처에서 모든 항목을 해제하고 적용하여 패치를 제거할 수 있습니다.


사용 중 발생하는 문제의 책임은 사용자에게 있습니다. 

동의하는 경우에만 사용해 주세요.

구버전의 사용자DB는 Korean 폴더를 삭제하지 않는한 계속 사용가능 합니다.

## 고유명사 구분의 한계

고유명사를 체크하거나 해제해도 적용되지 않는 단어들이 있습니다. 이유는 3가지입니다.

첫째, 내장DB에 없어서, 사용자DB로 수집된 경우 고유명사를 구분할 수 없습니다.

둘째, 내장DB에 있으나, 길드워2가 외부로 렌더링하는데 다른마커나 색상등 특수한 내부적인 문자열이 붙는경우.

셋째, 고유명사에도 있는 단어나 문구지만 다른카테고리에서 중복되는 경우.

최대한 구분하려고 시도했으나 완벽하지는 않다는 점 양해부탁드립니다.


## 업데이트

최신 실행 파일은 [Releases](https://github.com/motionsilse/gw2-korean-patch/releases)에서 받을 수 있습니다.

길드워2가 업데이트 되면, 한글패치는 잠시 해제 후 패치하시고 길드워2의 업데이트가 완료가 되면 다시 한글패치를 적용하시면 됩니다.

한글패치된 상태로 업데이트를 하게 되면 다운로드가 제대로 되지 않을 수 있습니다.
