# 길드워2 한글 패치

본 한글 패치는 비공식 사용자 제작 패치입니다.


사용자 DB 수집과 번역 적용은 켜고 사용하시는걸 추천드립니다.

길드워2의 입문자 : 전체 체크 모두 번역 추천 or 아이템 이름만 체크해제


길드워2 기존 플레이어거나 업적 및 아이템등 위키 사용을 위한 플레이어 : 전체 체크하고 고유명사만 체크해제

그밖에 사용자입맛에 맞게 설정해서 사용가능.



문제가 발생하면 패처에서 모든 항목을 해제하고 적용하여 패치를 제거할 수 있습니다.

이 패치는 완전한 한글화가 아니며, 번역 품질이 낮거나 일관성이 없을 수 있고, 잘못 번역된 문구가 포함될 수 있습니다.

사용 중 발생하는 문제의 책임은 사용자에게 있습니다. 

동의하는 경우에만 사용해 주세요.

구버전의 사용자DB는 Korean 폴더를 삭제하지 않는한 계속 사용가능 합니다.
