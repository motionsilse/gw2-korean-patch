# 길드워2 한글 패치

본 한글 패치는 비공식 사용자 제작 패치입니다.

이 패치는 완전한 한글화가 아니며, 전체 문자열을 번역하지 않습니다. 번역 품질이 낮거나 일관성이 없을 수 있고, 잘못 번역된 문구가 포함될 수 있습니다.

이 패치는 길드워2 한글화가 가능하다는 것을 보여주는 실험적 예시입니다. 향후 피드백 반영이나 추가 업데이트가 없을 수 있습니다.

사용 중 발생하는 문제의 책임은 사용자에게 있습니다. 문제가 발생하면 패처에서 모든 항목을 해제하고 적용하여 패치를 제거할 수 있습니다.

동의하는 경우에만 사용해 주세요.

## 다운로드

최신 실행 파일은 [Releases](https://github.com/motionsilse/gw2-korean-patch/releases)에서 받을 수 있습니다.

## 기반 프로젝트

이 패치는 [nourlie/gw2-cyrillic](https://github.com/nourlie/gw2-cyrillic) 기반으로 제작되었습니다.

길드워2 한글화 방법은 위 프로젝트를 참고하면 됩니다.
